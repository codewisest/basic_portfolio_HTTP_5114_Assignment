My Projects

Matching Cards Game

Udacity Nanodegree project requirement

This project was a project requirement for the Udacity’s Nanodegree scholarship I was awarded for a Google Africa challenge I took part in. This project is a memory game whereby pairs of cards are randomly positioned. User opens two to make one move. If they match, they remain open otherwise they return to their closed positions. The goal is to open all the cards in the least possible time frame. This project helped me to reinforce my DOM manipulation skills.
